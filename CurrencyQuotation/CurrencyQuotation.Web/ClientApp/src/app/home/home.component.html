Bienvenido al cotizador de Pesos Argentinos:

Este proyecto implementa una Web Api RESTful que retorna las cotizaciones de monedas en base al Peso Argentino usando la api cambio.today

Puede probar la aplicación web aquí

Tecnologías usadas

  • Backend: C#, ASP.NET MVC Core
  • Fronend: Angular

Ejempo de uso

Petición
  • Verbo http: GET
  • Url: https://[[hostname]]/Quotation/USD

    • Respuesta
    • updated: Fecha de la última actualización de la cotización
    • source: Moneda base de la cotización
    • target: Moneda a cotizar
    • value: Cotización de la moneda objetivo en base a la moneda base
    • target: Unidad de cotización
    • amount: Cotización

    Agregar monedas a cotizar en la aplicación web

    Para agregar monedas en la aplicación web, dirigirse al archivo config.json ubicado en la carpeta src\assets de la aplicación Angular

    Para ello necesitara saber el código de la moneda ISO 4217
